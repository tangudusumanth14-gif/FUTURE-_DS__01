Business Sales Performance Analytics

Project Overview

This project analyzes business sales data to identify revenue trends, top-selling products, category performance, and regional sales performance.

Tools Used

- Microsoft Excel
- GitHub

Dataset

Sample Business Sales Dataset (CSV)

Key Findings

- Electronics generated the highest revenue.
- Laptops and Mobiles were the top-selling products.
- South and North regions performed best.
- Stationery category generated the lowest revenue.

Recommendations

- Increase focus on Electronics products.
- Expand marketing efforts in high-performing regions.
- Improve low-performing categories using promotions.
