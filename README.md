# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer subscription data to identify churn patterns, retention trends, and customer behavior.

## Tools Used
- Microsoft Excel
- GitHub

## Dataset
Customer Churn Dataset (CSV)

## Key Findings
- Basic plan customers showed the highest churn rate.
- Premium customers had the highest retention.
- Customers with low usage hours were more likely to churn.
- Long-tenure customers remained loyal.

## Recommendations
- Improve engagement for Basic plan users.
- Offer loyalty rewards for long-term customers.
- Reduce churn through personalized retention campaigns.
- Improve customer support response times.
